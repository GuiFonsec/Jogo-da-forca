Bom dia/ tarde/ noite!
Este é o Jogo de adivinhação desenvolvido completamente na linguagem C.

A objetivo do jogo é que o jogador tente descobrir a fruta secreta escolhida randomicamente do banco de dados, chutando letra por letra.

Para rodar o jogo, é necessário abrir o prompt de comando do seu computador e procurar o diretório onde o jogo está armazenado ("cd" no caso do Windows, Exemplo: cd C:\Users\Administrador\Forca), e executar
o arquivo .exe escrevendo-o e apertando a tecla enter. (forca.exe)

É permitido, no final do jogo, adicionar uma nova fruta ao banco de dados (palavra.txt), mas cuidado para não adicionar frutas repetidas. 

IMPORTANTE! Todos os chutes tem que ser realizados com letra maiúscula.

Tenha um bom jogo!
